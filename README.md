# Ateliê de Costura - Controle e Agendamento

Este projeto é um sistema moderno, elegante e responsivo de qualificação e agendamento automático de visitas para um ateliê de costura, integrado ao **Supabase** (Banco de Dados + Auth) e pronto para ser hospedado no **Netlify**.


---

## 🚀 Como Executar o Projeto Localmente

1.  **Instalar dependências**:
    ```bash
    npm install
    ```

2.  **Iniciar o servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```

3.  **Configurar variáveis de ambiente**:
    Certifique-se de que o arquivo `.env.local` na raiz do projeto contém as chaves corretas obtidas no painel do Supabase.
